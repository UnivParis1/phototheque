[login-link]

Ezzel a taggel login linket helyezhetsz el mindenhol, automatikus átirányítással az adott oldalra.

Lehetőségek:
  • html (opcionális) amennyiben az értéke false a tag csak egy URL-t ad vissza HTML körítés nélkül.
  • text (opcionális) a link neve megadható, tartalmazhat[lang] tageket is
[login-link]

[login-link html=true text="log in[lang=hu]bejelentkezés[/lang]"]

[logged]

Csak egy szövegblokkot mutat azzal, hogy adott felhasználó be van-e jelentkezve, vagy nem.

[logged=true] Üdv újra [/logged]
[logged=false] Kérlek jelentkezz be [/logged]
