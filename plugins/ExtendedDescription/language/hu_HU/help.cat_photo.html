[photo id=xx]

Ez a tag lehetővé teszi, hogy bármilyen méretű fotót beszúrj.

Lehetőségek:
  • id a fotó azonosítója
  • album (opcionális) a szülőalbum azonosítója
  • size (opcionális) a fotó mérete, tól/től: (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (opcionális) false érték esetén a tag csak az fotó URL-jét adja vissza, formázott HTML nélkül.
  • link (opcionális) true érték esetén a fotó linkelve lesz a saját oldalához
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

Ugynúgy, mint a fenti, azzal a kivétellel, hogy a fotó véletlenszerűen kerül kiválasztásra a teljes galériából, vagy albumból.
