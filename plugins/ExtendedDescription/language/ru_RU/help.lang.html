Мультиязычные описания располагаются между тегами [lang=xx] и [/lang], где xx

  • код языка (например ru, en, fr, ...).

    [lang=en]Default description[/lang]
[lang=fr]Description en français[/lang]
[lang=de]Deutsche Beschreibung[/lang]

    Специальный язык default

    Описание по умолчанию будет использовано в случае, если отсутствует описание на выбранном текущим пользователем языке.
    Если [lang=default] не существует, все снаружи тега [lang] будет считаться описанием по умолчанию.

    [lang=default]Default description[/lang]
[lang=fr]Description en français[/lang]

// OR

Default description
[lang=fr]Description en français[/lang]

    Специальный язык all

    все внутри тега [lang=all] будет включено в описание независимо от выбранного текущим пользователем языка.
    Чаще всего это используется для включения HTML/JavaScript кода в описание.

    [lang=all]<p>[/lang]
  [lang=default]Default description[/lang]
  [lang=fr]Description en français[/lang]
  [lang=de]Deutsche Beschreibung[/lang]
[lang=all]</p>[/lang]
