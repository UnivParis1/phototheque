[photo id=xx]

Ten tag pozwala na wstawienie jednego zdjęcia, dowolnego rozmiaru.

Opcje:
  • id identyfikator zdjęcia
  • album (opcjonalna) identyfikator albumu z którego pochodzi zdjęcie
  • size (opcjonalna) rozmiar zdjęcia, dostępne: (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (opcjonalna) jeżeli nie ustawione na false to zwraca tylko URL zdjęcia, bez kodu HTML
  • link (opcjonalna) jeżeli ustawione na true to zdjęcie jest także odnośnikiem do strony zdjęcia
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

Identycznie jak powyżej, z tym że zdjęcie jest wybrane pseudolosowo z całej galerii lub z albumu album.

[random]

[random album=123 size=M html=yes link=yes]

[cat=xx]

Ten tag wstawia album do opisu, gdzie xx jest identyfikatorem albumu.
