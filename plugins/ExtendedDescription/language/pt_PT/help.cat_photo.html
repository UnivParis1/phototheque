[photo id=xx]

Esta etiqueta permite inserir uma imagem de qualquer tamanho que você queira.

Opções:
  • id identificador da foto
  • album (opcional) o identificador do album pai
  • size (opcional) o tamanho da foto, entre (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (opcional) se definido para false retorna a URL da foto, sem HTML
  • link (opcional) se definido para true a foto é incluida num link para a página da foto
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

A Mesma coisa que acima, exceto que a foto é escolhida aleatoriamente em random dentro album.

[random]

[random album=123 size=M html=yes link=yes]

tag [cat=xx]

Esta etiqueta insere um album na descrição, onde xx é o identificador do album.
