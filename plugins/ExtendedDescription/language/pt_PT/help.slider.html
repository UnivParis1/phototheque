[slider album=xx]

Esta etiqueta permite inserir um slideshow.

Opções: (você deve definir album OR list)
  • album (opcional)fonte album
  • nb_images (opcional)máximo numero de fotos para exibir no slideshow
  • random (opcional) escolhe fotos aleatoriamente no álbum
  • list (opcional) uma lista de fotos com ids separados por vírgula
  • size (opcional) o tamanho das fotos, entre (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • speed (opcional) Velocidade de transição
  • title (opcional) mostra título da foto
  • effect (opcional) efeito de transição (ver NivoSlider doc)
  • arrows (opcional) mostra setas de navegação
  • elastic (opcional) adapta o tamanho do slideshow para cada foto
  • control (opcional) exibe os controles de navegação (true, false, thumb)
  • thumbs_size (optional) thumbnails size in pixel if control=thumb
[slider album=123]

[slider list=46,47,52]

[slider album=123 nb_images=10 random=false size=M speed=3 title=false effect=fade arrows=true elastic=false control=true thumbs_size=80]
