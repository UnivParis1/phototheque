[login-link]

This tag allows to insert everywhere a login link with automatic redirection to the current page.

Options:
  • html (optional) if set to false the tag only returns the login URL, without HTML
  • text (optional) link label, can contain [lang] tags
[login-link]

[login-link html=true text="log in[lang=fr]connectez-vous[/lang]"]

[logged]

Display a block of text only if the current user is logged or not.

[logged=true] Welcome back [/logged]
[logged=false] Please log in [/logged]
