[photo id=xx]

此tag标签用于在描述中插入一张任意尺寸的图片：

选项：
  • id 图片id
  • album (可选) 当前相册id
  • size (可选) 图片尺寸，有 (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (可选) 如果设为 false 则tag标签仅给出图片的URL，不包含HTML
  • link (可选) 如果设为 true 则点击图片可进入该图片所在页面
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

同上，只是图片是从整个图库或album中随机抽选的。

[random]

[random album=123 size=M html=yes link=yes]

[cat=xx]

此tag标签可在描述中插入一个相册，xx为相册id。
