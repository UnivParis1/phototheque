[login-link]

Цей теґ дозволяє вставляти всюди посилання для входу з автоматичним перенаправленням на поточний вік.

Опції:
  • html - (опціонально) якщо встановити false теґ повертає тільки логін URL, без HTML
  • text - (опціонально) посилання, що може містити ознаки мови [lang] 
[login-link]

[login-link html=true text="log in[lang=fr]connectez-vous[/lang]"]

[logged]

Показувати блок тексту тільки якщо поточний користувач зареєструвався чи ні.

[logged=true] Welcome back [/logged]
[logged=false] Please log in [/logged]
