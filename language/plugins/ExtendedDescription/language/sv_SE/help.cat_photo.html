[photo id=xx]

Med den här taggen kan du infoga en bild av vilken storlek du vill.

Val:
  • id fotots unika ID
  • album (valfritt) albumets unika ID
  • size (valfritt) Önskad storlek på bilden, från (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (valfritt) Om inställd till false returnerar taggen endast webbadressen till bilden, utan HTML-kod
  • link (valfritt) Om inställd till true blir bilden även en länk till den aktuella bildsidan

[photo id=46 album=22 size=M html=true link=true]
