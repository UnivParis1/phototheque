[photo id=xx]

This tag allows you to insert one photo of any size you want.

Options:
  • id the photo identifier
  • album (optional) the parent album identifier
  • size (optional) the size of the photo, from (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (optional) if set to false the tag only returns the URL of the photo, without HTML
  • link (optional) if set to true the photo is wrapped into a link to the photo page
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

Same thing as above, except that the photo is chosen randomly from the whole gallery or from album.

[random]

[random album=123 size=M html=yes link=yes]

[cat=xx]

This tag inserts an album in the description, where xx is the album identifier.
