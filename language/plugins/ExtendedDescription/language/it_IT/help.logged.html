[login-link]

Questo tag permette di inserire ovunque un link di accesso con reindirizzamento automatico all'attuale età.

Opzioni:
  • html (opzionale) se è impostato a false il tag restituisce solo l'URL di accesso, senza HTML
  • testo etichetta di collegamento (opzionale), può contenere tag [lang]
    • [login-link]

[login-link html=true text="log in[lang=fr]connectez-vous[/lang]"]

    [logged]

    Visualizzare un blocco di testo solo se l'utente corrente è connesso o no.

    [logged=true] Ben tornato [/logged]
[logged=false] Prego fai log in [/logged]
