[login-link]

Deze tag laat toe om overal een login-link te plaatsen met een automatische doorverwijzing naar de huidige leeftijd.

Opties:
  • html (optional) if set to false the tag only returns the login URL, without HTML
  • text (optional) link label, can contain [lang] tags
[login-link]

[login-link html=true text="log in[lang=fr]connectez-vous[/lang]"]

[logged]

Plaats een tekstblok als de gebruiker is ingelogd of niet.

[logged=true] Welkom terug [/logged]
[logged=false] Gelieve in te loggen [/logged]
