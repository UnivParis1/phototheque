[photo id=xx]

Этот тэг позволяет Вам вставить одно изображение любого размера, который пожелаете.

опции:
  • id идентификатор изображения
  • album (не обязательно) идентификатор альбома, содаржещего нужное изображение
  • size (не обязательно) размер изображения, из диапазона (SQ, TH, XXS, XS, S, M, L, XL, XXL)
  • html (не обязательно) если указать false, то тэг указывает только URL изображения, без HTML
  • link (не обязательно) если указать true, то изображение становится ссылкой на картинку
[photo id=46]

[photo id=46 album=22 size=M html=true link=true]

[random album=xx]

То же самое, что и выше, но только изображение выбирается случайным образом из альбома album.

[random]

[random album=123 size=M html=yes link=yes]

[cat=xx]

Этот тэг вставляет альбом в описание, при этом xx это идентификатор альбома.
