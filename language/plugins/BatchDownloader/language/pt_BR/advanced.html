$conf['batch_download_max_elements']
Valor máximo de O número máximo de fotos por set para baixar deslizante.
O padrão é 1000.
$conf['batch_download_max_size']
Valor máximo do Tamanho máximo de cada arquivo deslizante.
O padrão é 500.
$conf['batch_download_additional_ext']
Array contendo as extensões de arquivo para download (existe distição entre maiúsculas e minúsculas).
O padrão é $conf['picture_ext'].
$conf['batch_download_file_pattern']
Padrão para nomes de arquivos no arquivo.
Campos disponíveis são %id%, %filename%, %author%, %dimensions%.
As funções disponíveis são $escape(xxx), $upper(xxx), $lower(xxx), $strpad(xxx, z, c).
O valor padrão é %id%_%filename%_%dimensions%.
Exemplo: $strpad(%id%,4,0)_$escape(%filename%)_$lower(%author%)_%dimensions%.
